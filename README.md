# 📊 PappiGestor

Um sistema de gestão desenvolvido em **Next.js 16**, com **Tailwind CSS 4**, **Supabase** e diversos componentes da **Radix UI**.  
O objetivo é oferecer uma plataforma moderna, responsiva e escalável para gerenciamento de dados e processos internos.

---

## 🚀 Tecnologias

- [Next.js 16](https://nextjs.org/) com suporte a **Turbopack**
- [React 19](https://react.dev/)
- [Tailwind CSS 4](https://tailwindcss.com/) + [tailwindcss-animate](https://github.com/jamiebuilds/tailwindcss-animate)
- [Supabase](https://supabase.com/) para autenticação e banco de dados
- [Radix UI](https://www.radix-ui.com/) para componentes acessíveis
- [Lucide Icons](https://lucide.dev/) para ícones
- [Recharts](https://recharts.org/) para gráficos
- [XLSX](https://github.com/SheetJS/sheetjs) e [jsPDF](https://github.com/parallax/jsPDF) para exportação de dados
